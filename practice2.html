Image

we can select image which is present in our system .when we open image at top we can see folder tab. when we click on that we can see the path.copy that path and paste in src and give \ and enter name of the image.
we can also adjust the size of the image by using height & width attributes.
we can also use any website images.
we can apply borders to image by using border attribute.
we can align the image to right using align attribute.
if we use left alignment then the image will be at left and the para will come from bottom to right of the image.
If we want to get the image to center use center tag.
we can use image as background to the page by using background attribute at body.
